# Home Manager

A personal home management application for organizing household activities.

## Features

### Dashboard
Get a quick overview of your home with today's priorities and key metrics at a glance.

### Chores Management
- Track daily, weekly, and recurring household tasks
- Calendar view to see what's due when
- Assignment tracking for family members
- Completion history and statistics

### Finances
- Expense tracking with categorization
- Income recording and management
- Monthly budget planning
- Savings goals with progress tracking
- Subscription management
- Visual charts for financial overview

### Food Planning
- Weekly menu planning
- Shopping list generation
- Pantry inventory tracking

### People
- Manage household members and profiles
- Track individual contributions and assignments

## Usage

This is a single-user personal application. Access is restricted to existing users only.


## Privacy

This application handles personal household data. Environment files containing sensitive configuration are excluded from version control.
